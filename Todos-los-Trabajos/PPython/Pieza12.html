Avance de Proyecto:





El script Python utiliza la biblioteca PyMySQL para conectarse a una base de datos MySQL local y realizar una serie de operaciones SQL. Comienza estableciendo conexiones a la base de datos utilizando distintos usuarios y contraseñas para diferentes tareas.

Luego, se ejecutan comandos SQL para crear tablas en las bases de datos mantenimiento y horadeentrada, definiendo la estructura de las mismas con columnas específicas para almacenar información relacionada con el mantenimiento de motores, sensores y seleccionadores, así como registros de la hora de entrada de los trabajadores.

Para poblar estas tablas con datos simulados, se definen funciones que generan valores aleatorios para el estado de los dispositivos y si los trabajadores llegaron a tiempo. Estos datos se insertan en las tablas utilizando instrucciones SQL de inserción.

Además de la inserción de datos, el script otorga y revoca permisos de SELECT a diferentes usuarios para acceder a tablas específicas en las bases de datos mantenimiento y horadeentrada. Esto se realiza mediante comandos SQL GRANT y REVOKE.

Finalmente, se cierran todas las conexiones a la base de datos una vez que se han completado las operaciones. En resumen, el código automatiza la creación de tablas, la inserción de datos y la gestión de permisos en una base de datos MySQL utilizando Python y PyMySQL.
