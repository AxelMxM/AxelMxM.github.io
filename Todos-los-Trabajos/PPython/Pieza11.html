Crear una base de datos que contenga una tabla de 120 valores aleatorios entre 7 y 10 las cuales representan las calificaciones de 6 materias de 20 alumnos de un salón de Universidad.





Al final deberás obtener el promedio de cada alumno y si es mayor a 8 regresar una lista con su nombre y Decisión.

La decisión puede ser Aprobado o No Aprobado: Este programa se conecta a una base de datos MySQL localmente usando la biblioteca pymysql.

Si no existe, crea una tabla llamada calificaciones con campos para el ID del alumno, nombre y calificaciones en varias materias.

Genera aleatoriamente calificaciones para 20 alumnos en diferentes materias, inserta estos datos en la tabla y luego calcula el promedio de las calificaciones de cada alumno. Si el promedio es mayor a 8, se marca como "Aprobado"; de lo contrario, se marca como "No Aprobado".

Luego imprime en pantalla los nombres de los alumnos aprobados junto con sus promedios y decisiones de aprobación. Finalmente, cierra la conexión con la base de datos.
