Programación PLC





Considere que se va a desarrollar una aplicación que se controlará por medio de un PLC. Se tiene un estacionamiento para 10 automóviles el cual cuenta con una de entrada y una de salida, en estos puntos se tienen instaladas plumas para controlar el paso de vehículos y cuentan con sensores para detectar la posición de la pluma, es decir, si se encuentra abierta o cerrada; así como también, se tiene dos sensores para verificar la existencia de automóvil antes y después de la pluma.

Para activar la barrera de entrada se cuenta con un botón para solicitar el boleto de acceso al estacionamiento y se cuenta con un lector de código de barras para permitir la apertura de la pluma de salida, ya que se efectúa el pago.

Se deberán controlar las plumas de acceso y salida de la siguiente forma:

Si el estacionamiento tiene capacidad para recibir autos este permitirá que la pluma de entrada se abra, de estar lleno no se abrirá.

La pluma de entrada podrá abrirse exclusivamente si se detecta presencia de automóvil de lo contrario no se entregará boleto y no se abrirá la barrera. Algo similar ocurre con la pluma de salida, pero esta se activa si el código de barras detecta que ya se efectúo el pago.

Si está lleno el estacionamiento se tendrá una lámpara que así lo indique, y si hay cupo se tendrá otra que también lo haga saber.

Se deberá de controlar el cambio de giro del motor que opera las plumas. Se debe tener la precaución de que la pluma no vaya a golpear un vehículo al momento de cerrase.
